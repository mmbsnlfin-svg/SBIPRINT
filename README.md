# Excel Splitter & PDF Generator

This Streamlit app selects output columns by **header name**, not by column letter.

## Fixed output columns

- Sr. No. (generated by the app)
- ACCOUNT _NUM
- LC ID
- Bill From
- Bill To
- Days
- Branch Code
- Branch Name
- C Type
- Port BW
- Annual Recurring Charges
- Quarterly Charges
- NTU Chg /Modem Chg
- NOFN charges
- IDR / Submarine Charges
- Total Quarterly charges Gross
- GST 18%
- Net Payable After Tax
- GST STATE
- Parent BA
- PO NO
- PO Date

The app automatically excludes PO, Type, CGST 9%, SGST 9%, Remarks, Loopback and WAN IP.

## Formula handling

The following formulas are rebuilt after the output columns are created:

- Days = Bill To - Bill From + 1
- Quarterly Charges = Annual Recurring Charges / 4
- Total Quarterly charges Gross = Quarterly Charges + NTU + NOFN + IDR/Submarine
- GST 18% = Total Quarterly charges Gross × 18%
- Net Payable After Tax = Total Quarterly charges Gross + GST 18%

This prevents broken references after unwanted source columns are excluded.

## Deployment

Upload these files to the root of a GitHub repository:

- `app.py`
- `requirements.txt`
- `packages.txt`

Deploy `app.py` through Streamlit Community Cloud.
